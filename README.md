## Installation

```
git clone https://github.com/almasjss/StockManageHackathon.git
Download XAMPP
Add package to "htdocs"
Open phpmyadmin
Create "userinfo" database
Import "userinfo.sql"
```

## Usage
```
localhost/StockManageHackathon
```
